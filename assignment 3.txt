1.How arithmetic operators and concatenation works in browser

1. Arithmetic Operators in the Browser (JavaScript)

JavaScript supports standard arithmetic operators:

|Operator| Meaning             |Example |Result|
| ------ | ------------------- | ------ | ---- |
| +      | Addition            | 5 + 3  | 8    |
| -      | Subtraction         | 5 - 3  | 2    |
| *      | Multiplication      | 5 * 3  | 15   |
| /      | Division            | 10 / 2 | 5    |
| %      | Modulus (remainder) | 10 % 3 | 1    |
| **     | Exponent            | 2 ** 3 | 8    |

 Example:

```js
let a = 10;
let b = 5;

console.log(a + b); // 15
console.log(a * b); // 50
```


 2. Concatenation in the Browser

 (a) String Concatenation using `+`

When **`+` is used with strings**, JavaScript **joins them** instead of adding.

```js
"Hello" + " World"
// Result: "Hello World"
```

 (b) Mixing Numbers and Strings

If either operand is a string, JavaScript converts the other operand to a string and concatenates.

```js
"5" + 3   // "53"
3 + "5"   // "35"
```

 This is called type coercion.


 3. Order of Execution (Left to Right)

JavaScript evaluates expressions from left to right.

```js
5 + 3 + "2"   // "82"
```

Steps:

1. `5 + 3 = 8`
2. `8 + "2" = "82"` (number â†’ string)

```js
"5" + 3 + 2   // "532"
```

Steps:

1. `"5" + 3 = "53"`
2. `"53" + 2 = "532"`

---

 4. Subtraction, Multiplication, Division with Strings

Other operators force numeric conversion.

```js
"10" - 2   // 8
"10" * 2   // 20
"10" / 2   // 5
```

 Only `+` performs concatenation.



 5. Browser Example (HTML + JS)

```html
<script>
  let x = "10";
  let y = 5;

  document.write(x + y); // 105 (concatenation)
  document.write("<br>");
  document.write(x - y); // 5 (arithmetic)
</script>
```
